Quadratic equation

Introduction

Before we introduce the formulas for solving quadratic equations, let's recall how we have been solving simple equations with one unknown variable x raised to the first power. These were equations similar to the one below.

a) \$5x=x-4 \$

b) \$-2(x-7)+5x=3x+9\$

c) \$x-10=x+7 \$

[[[AlertCounter]]]

These types of equations can be reduced to the general form \$ax + b = 0\$ or \$ax = b\$, where \$x\$ is the unknown variable we are solving for, and \$a\$ and b are coefficients. We can solve such an equation. Assuming a ≠ 0, we have the solution:

\[ x = \frac{-b}{a} \]

If a = 0, the equation reduces to b = 0 and becomes independent of the value of x. For example, if b is also zero, then there are infinitely many solutions.

To summarize, a linear equation with one unknown has a single solution when a ≠ 0. If a = 0, it has infinitely many solutions when b = 0, or no solution at all (i.e., it's inconsistent) when b ≠ 0.

Solve equation \$ -2(x-7)+5x=3x+9\$
\[\begin{aligned} -2(x-7)+5x&=3x+9 \\ -2x+14+5x&=3x+9 \ /-14 \\ 3x&=3x-5 \ /-3x \\ 0x&=-5 \\ 0&=-5 \end{aligned}\]

The equation has no solution. The coefficient \$a=0\$ and \$b=5\$.

Simple Quadratic Equations \$ax^2=b\$

Quadratic equations are equations in which the unknown appears in the second power.

We can solve quadratic equations by factoring out a common factor or by using the special product formulas:

\[ \begin{aligned} a^2 - b^2 &= (a + b)(a - b) \\ (a \pm b)^2 &= a^2 \pm 2ab + b^2 \end{aligned} \]
Solving the equation \$x^2=4x\$ by factoring out \$x\$: \[\begin{aligned} x^2 &= 4x \\ x^2 - 4x &= 0 \\ x(x-4) &= 0 \end{aligned}\] We get two solutions: \[\begin{aligned} x = 0 &\vee x - 4 = 0 \\ x = 0 &\vee x = 4 \end{aligned}\]
Solving the equation \$7x^2 + 35x = 0\$ by factoring out \$7x\$: \[\begin{aligned} 7x^2 + 35x &= 0 \\ 7x(x+5) &= 0 \end{aligned}\] We get two solutions: \[\begin{aligned} 7x = 0 &\vee x + 5 = 0 \\ x = 0 &\vee x = -5 \end{aligned}\]
Solving the equation \$-3x^2 = 24x\$ by factoring out \$-3x\$: \[\begin{aligned} -3x^2 &= 24x \\ -3x^2 - 24x &= 0 \\ -3x(x+8) &= 0 \end{aligned}\] We get two solutions: \[\begin{aligned} x \in \{ 0, -8 \} \end{aligned}\]
Solving the equation \$x^2 = 9\$ using the difference of squares formula \$a^2 - b^2 = (a + b)(a - b)\$: \[\begin{aligned} x^2 &= 9 \\ x^2 - 9 &= 0 \\ x^2 - 3^2 &= 0 \\ (x + 3)(x - 3) &= 0 \end{aligned}\] We get two solutions: \[\begin{aligned} x + 3 = 0 &\vee x - 3 = 0 \\ x = -3 &\vee x = 3 \end{aligned}\]
Solving the equation \$x^2 = 8\$ using the difference of squares formula \$a^2 - b^2 = (a + b)(a - b)\$: \[\begin{aligned} x^2 &= 8 \\ x^2 - 8 &= 0 \\ x^2 - (\sqrt{8})^2 &= 0 \\ (x + \sqrt{8})(x - \sqrt{8}) &= 0 \end{aligned}\] We get two solutions: \[\begin{aligned} x + \sqrt{8} = 0 &\vee x - \sqrt{8} = 0 \\ x = -\sqrt{8} &\vee x = \sqrt{8} \end{aligned}\]
Solve the equation \$16x^2 = 5\$
\[\begin{aligned} 16x^2 &= 5 \\ x^2 &= \frac{5}{16} \\ x^2 - \frac{5}{16} &= 0 \\ x^2 - \left(\sqrt{\frac{5}{16}}\right)^2 &= 0 \\ x^2 - \left(\frac{\sqrt{5}}{\sqrt{16}}\right)^2 &= 0 \\ x^2 - \left(\frac{\sqrt{5}}{4}\right)^2 &= 0 \\ \left(x + \frac{\sqrt{5}}{4}\right) \left(x - \frac{\sqrt{5}}{4}\right) &= 0 \end{aligned}\] There are two solutions: \[\begin{aligned} x = -\frac{\sqrt{5}}{4} &\vee x = \frac{\sqrt{5}}{4} \end{aligned}\]
Solving the equation \$x^2 + 6x + 9 = 0\$ using the special product formula: \$(a + b)^2 = a^2 + 2ab + b^2\$. \[\begin{aligned} x^2 + 6x + 9 &= 0 \\ x^2 + 2 \cdot x \cdot 3 + 3^2 &= 0 \\ (x + 3)^2 &= 0 \end{aligned}\] We get one solution: \[\begin{aligned} x + 3 &= 0 \\ x &= -3 \end{aligned}\]

As seen from the examples, a quadratic equation can have two solutions.

Quadratic Equation in General Form

The quadratic equation in general form: \[ ax^2 + bx + c = 0 \] where \$x\$ is the unknown, and \$a\$, \$b\$, and \$c\$ are the coefficients, with \$a \neq 0\$.

The quadratic equation can have at most two solutions, which can be calculated using the following formulas.

The formula for the discriminant (delta) of the quadratic equation:

\[ \Delta = b^2 - 4ac \]

The quadratic equation has real solutions (\$\mathbb{R}\$) when \$\Delta \ge 0\$:

\[ x_1 = \frac{-b + \sqrt{\Delta}}{2a} \quad x_2 = \frac{-b - \sqrt{\Delta}}{2a} \]

To solve the quadratic equation:

1) Calculate the discriminant \$\Delta = b^2 - 4ac\$.

2) If \$\Delta < 0\$, there are no solutions.

3) If \$\Delta = 0\$, there is one solution.

4) If \$\Delta > 0\$, there are two solutions:

\[ x_1 = \frac{-b + \sqrt{\Delta}}{2a} \quad x_2 = \frac{-b - \sqrt{\Delta}}{2a} \]
Solve the equation \$x^2 + 6x + 9 = 0\$

The coefficients are \$a = 1, b = 6, c = 9\$.

\[\begin{aligned} \Delta = 6^2 - 4*1*9 = 36 - 36 = 0 \end{aligned}\] There is one solution: \[\begin{aligned} x = \frac{-6}{2*1} = \frac{-6}{2} = -3 \end{aligned}\]
Solve the equation \$x^2 + 5 = 6x\$

Rearrange the equation to the general form.

\[ x^2 - 6x + 5 = 0 \]

The coefficients are \$a = 1, b = -6, c = 5\$.

\[\begin{aligned} \Delta = (-6)^2 - 4*1*5 = 36 - 20 = 16 > 0 \end{aligned}\] There are two solutions, and we have \$\sqrt{\Delta} = 4\$: \[\begin{aligned} x_1 = \frac{-(-6) + 4}{2*1} = \frac{6 + 4}{2} = \frac{10}{2} = 5 \end{aligned}\] \[\begin{aligned} x_2 = \frac{-(-6) - 4}{2*1} = \frac{6 - 4}{2} = \frac{2}{2} = 1 \end{aligned}\]
Solve the equation \$-2x^2 + 15x + 8 = 0\$

To simplify the calculations, we can multiply both sides of the equation by \$-1\$.

\[\begin{aligned} -2x^2 + 15x + 8 &= 0 \ /*(-1) \\ 2x^2 - 15x - 8 &= 0 \end{aligned}\]

The coefficients are \$a = 2, b = -15, c = -8\$.

\[\begin{aligned} \Delta = (-15)^2 - 4*2*(-8) = 225 + 64 = 289 > 0 \end{aligned}\] There are two solutions, and we have \$\sqrt{\Delta} = 17\$: \[\begin{aligned} x_1 = \frac{-(-15) + 17}{2*2} = \frac{15 + 17}{4} = \frac{32}{4} = 8 \end{aligned}\] \[\begin{aligned} x_2 = \frac{-(-15) - 17}{2*2} = \frac{15 - 17}{4} = \frac{-2}{4} = -\frac{1}{2} \end{aligned}\]
