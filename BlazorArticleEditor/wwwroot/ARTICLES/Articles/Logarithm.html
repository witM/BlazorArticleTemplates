Logarithm

Calculating Logarithms

The logarithm of a number \$b > 0\$ with base \$a > 0\$ and \$a \neq 1\$ is defined as the number \$x\$ such that \$a^x = b\$:

\[ \log_a(b) = x \Leftrightarrow a^x = b \]

The number \$b\$ is called the antilogarithm, and the number \$a\$ is called the base of the logarithm.

To calculate a logarithm \$\log_a(b)\$, we look for the number \$x\$ such that raising \$a\$ to the power of \$x\$ equals \$b\$.

A common logarithm is a logarithm with base \$a = 10\$, and is written in short form as: \$\log_{10}(b) = \log(b)\$.

Below are examples of calculating logarithms from the definition.

Calculate \$\displaystyle{\log_2(8)}\$

Let \$\log_2(8) = x\$, then we are looking for such an \$x\$ that:

\[ 2^x = 8 \]

We try to express both sides using the same base.

\[ 2^x = 2^3 \]

From this we get:

\[ x = 3 \]

So, \$\log_2(8) = 3\$.

Calculate \$\log_3(9)\$

Let \$\log_3(9) = x\$, then we are looking for such an \$x\$ that:

\[ 3^x = 9 \]

We try to express both sides using the same base.

\[ 3^x = 3^2 \]

So

\[\begin{aligned} x = 2 \end{aligned}\]

We get \$\log_3(9) = 2\$.

Calculate \$\displaystyle{\log_{16}\left(\frac{1}{4}\right)}\$

Let \$\log_{16}\left(\frac{1}{4}\right) = x\$, then we are looking for such an \$x\$ that:

\[ 16^x = \frac{1}{4} \]

We try to express both sides using the same base.

\[\begin{aligned} \left(4^2\right)^x &= 4^{-1} \end{aligned}\]

Using the power rule, we multiply the exponents:

\[\begin{aligned} 4^{2x} = 4^{-1} \end{aligned}\]

So

\[\begin{aligned} 2x &= -1 \\ x &= -\frac{1}{2} \end{aligned}\]

We get \$\displaystyle{\log_{16}\left(\frac{1}{4}\right) = -\frac{1}{2}}\$.

Calculate \$\displaystyle{\log_{\sqrt{2}}(64)}\$

Let \$\log_{\sqrt{2}}(64) = x\$, then we are looking for such an \$x\$ that:

\[\begin{aligned} \left(\sqrt{2}\right)^x &= 64 \\ \left(2^{\frac{1}{2}}\right)^x &= 2^6 \\ 2^{\frac{1}{2}x} &= 2^6 \end{aligned}\]

So

\[\begin{aligned} \frac{1}{2}x &= 6 \\ x &= 12 \end{aligned}\]

We get \$\log_{\sqrt{2}}(64) = 12\$.

Calculate \$\displaystyle{\log\left(\frac{1}{1000}\right)}\$

Let \$\log\left(\frac{1}{1000}\right) = x\$, then we are looking for such an \$x\$ that:

\[\begin{aligned} 10^x &= \frac{1}{1000} \\ 10^x &= 1000^{-1} \\ 10^x &= \left(10^3\right)^{-1} \\ 10^x &= 10^{-3} \end{aligned}\]

So

\[\begin{aligned} x = -3 \end{aligned}\]

We get \$\log\left(\frac{1}{1000}\right) = -3\$.

Changing the Base of a Logarithm

The following formula allows us to convert a logarithm with base \$a\$ into an expression using a logarithm with base \$c\$.

\[ \log_a(b) = \frac{\log_c(b)}{\log_c(a)} \]

We can use this formula to approximate an irrational logarithm using a calculator. Most calculators compute base-10 (common) logarithms.

Calculate \$\displaystyle{\log_7(9)}\$

Using the base change formula, we convert the base to \$10\$:

\[\begin{aligned} \log_7(9) = \frac{\log_{10}(9)}{\log_{10}(7)} \end{aligned}\]

Using a calculator, we approximate \$\log_{10}(9)\$ and \$\log_{10}(7)\$ to 4 decimal places:

\[\begin{aligned} \frac{\log(9)}{\log(7)} \approx \frac{0.9542}{0.8451} \end{aligned}\]

Then, we calculate the final result (to 4 decimal places):

\[\begin{aligned} \frac{0.9542}{0.8451} = 1.1291 \end{aligned}\]

So, \$\displaystyle{ \log_7(9) \approx 1.1291 }\$.
